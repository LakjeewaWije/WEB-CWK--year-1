STUDENT 2 Individual Information

INFORMATION PAGE | DESIGN ISSUES

Student Name : Yazeen Thariq (ID : w1654091)

♿ Accessibility

  • All text content is readable and very easy to understand. And the font that i used is very professional and used throughout all the web pages.
  • Each page allows direct access to all pages in the web sites through the fixed navigation bar. In the Navigation Every Anchor has a larger area to user to click on.
  • Every link has it's hover effect so that the user can easily identify which one is going to select.
  • Text alternatives for Images, in the form of alt tags.
  • Each links works perfectly and they are relative to one on one links, so the user will not have trouble changing the entire website folder to another location.
  • Anchor tags for pages which are longer than the browser size.

✎ Typography

  • Main Font used is Open Sans - sans-serif and the secondary font is Roboto Bold.
  • Fonts are imported/provided through the google's content delivery network, because the loading times are much faster that added libraries.
  • Font sizes are given in order to avoid strain for the user, and to catch the attraction towards the necessary parts.
  • For every heading I used same font-size and font color to keep the consistancy.
  • For every sub-heading I used same font-size and font color to keep the consistancy.
  • Every paragraph's font weight is light & super readable. Used font size is 16 pixels  which the recommended by W3C.

☼ Color-Contrast / Balance

  • Appropriate colors are given to the necessary parts and sections to catch the user attraction.
  • Well balanced color contrasts (Text contents and background) spreads through the website.
  • Even the hover effects has it's attractive colors in the similar pattern matching with the theme.
  • Navigation Bar and the footer contains Light grey color background while the text remain white color. And the background has a small transparency, inorder to have some effect.
  • Fonts are in dark grey color, so it can be visible sharply with the white backgrounds in the content area.
  • Fonts are in white color in the navigation bar, so it can be visible sharply with the darker grey background. When user hover over a anchor tag it will be change to Mountain Meadow. And when if user hover over a anchor within an anchor color will change to Spring Green.
  • In the smaller titles in the content area I have used color Atoll to make it more attractive to the user.

◉ Gastalt Principles for grouping


◈ Similarity

  • The website has a well visible similarity, so the user can easily identify the grouped parts. For example, the buttons in the Comments Page look similar with the same styles and the same effects(hover) for other webpages. Also the dropdown lists in the navigation bar looks similar to the main navigation lists in the navigation bar.

◈ Proximity

  • A flow of symmetrical balance runs through all pages in our webPages. For example, the Wall_1_content in the main page has the same styles in similar wall_2_contents so that it appears like a group.

◈ Closure

  • Our main font Open Sans is a good Example for Closure. The human brain will find it a very attractive font even if itâ€™s full of broken lines. Another example is the buttons in the comments page. Even the button in the comments page centered relatively, still it looks as it's grouped together.

◈ Continuance

  • This principle is mostly used featured-image section and the page contents of the main page, as it catches user attraction at the beginning itself when the page Is loaded. Also in the "Thank You" part in the comments-result Page same as the comments page, it's also catching the user attraction directly.
